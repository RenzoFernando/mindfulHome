# mindfulHome

Aplicación web full-stack desarrollada con **Python**, **JavaScript** y **CSS**. El proyecto está dividido en un backend y un frontend, con el objetivo de ofrecer una experiencia web organizada, funcional y fácil de mantener.

## Descripción

`mindfulHome` es una aplicación web que integra un backend en Python con un frontend desarrollado en tecnologías web. El proyecto incluye mejoras en el manejo de errores de la API, formato de valores monetarios y ajustes en la interfaz de usuario.

## Estructura del proyecto

```text
mindfulHome/
├── mindfulhome_backend/     # Backend de la aplicación
├── mindfulhome_frontend/    # Frontend de la aplicación
└── README.md
```

## Tecnologías utilizadas

* Python
* JavaScript
* CSS
* HTML
* API REST

## Características principales

* Separación entre backend y frontend.
* Manejo mejorado de errores en la API.
* Formato de moneda más claro para el usuario.
* Interfaz web ajustada para una mejor experiencia.
* Organización modular del proyecto.

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/RenzoFernando/mindfulHome.git
cd mindfulHome
```

## Backend

Ingresa a la carpeta del backend:

```bash
cd mindfulhome_backend
```

Crea y activa un entorno virtual:

```bash
python -m venv venv
```

En Windows:

```bash
venv\Scripts\activate
```

En Linux/macOS:

```bash
source venv/bin/activate
```

Instala las dependencias si el proyecto cuenta con un archivo de requisitos:

```bash
pip install -r requirements.txt
```

Ejecuta el backend según el archivo principal del proyecto.

## Frontend

Ingresa a la carpeta del frontend:

```bash
cd mindfulhome_frontend
```

Instala las dependencias si el proyecto usa Node.js:

```bash
npm install
```

Ejecuta el frontend:

```bash
npm start
```

## Uso

Una vez ejecutados el backend y el frontend, abre la aplicación en el navegador usando la dirección local indicada por el servidor.
